    • 「On and Off」(ABC228 A問題)

    Difficulty : 55 , solved : 5570/6190

    問題文へのリンクはこちら

    問題文

    高橋君は、毎日 \(S\) 時 \(0\) 分に部屋の電気をつけ、毎時 \(T\) 時 \(0\) 分に消します
    電気をつけている間に日付が変わることもあります。

    \(X\) 時 \(30\) 分に部屋の電気が付いているかどうか判定して下さい。

    制約

    • \(0 \le S,T,X \le 23\)
    • \(S \ne T\)
    • 入力は全て整数である。

    解説

    この問題は,A問題の中でも比較的Difficultyが高い問題となりました.

    WAになるケース

    問題文の条件から,このような条件分岐を考えた方もいると思います。

    1. S,T,X = map(int,input().split())
    2. if S <= T < X:
    3.   print("Yes")
    4. else:
    5.   print("No")

    残念ながらこのソースコードではWAになります.理由としては問題文の条件から常に \(S \lt T\) が成り立っているとは言えず,
    \(T \lt S\) の場合に条件分岐が変になってしまう等の理由が挙げられます.
    では,どのようにしてこの問題を解くと良いのでしょうか？

    \(S \lt T\) の場合

    \(S\) 時 \(30\) 分に電気が付いていて, \(T\) 時 \(30\) 分には電気は消えている事が分かります. 従って, \(S \le X \lt T\) であるかどうかを判定すれば良いです.

    \(T \lt S\) の場合

    この場合は若干考察が複雑になります.例えば,6時から4時まで電気がついている,という状況は,少々分かりづらいかもしれません.
    ここでは,あえて「電気がついていない」時間で考えてみましょう.
    「電気がついていない」時間は問題文の条件から, \(T\) 時 \(0\) 分以降 \(S\) 時 \(0\) 分以降であることが分かります.
    従って, \(T \le X \lt S\) でないかどうかを判定すれば良いです.

    以上の条件分岐により,この問題を解くことが出来ました.
    1. s,t,x = map(int,input().split())
    2. if s < t:
    3.     if s <= x < t:
    4.         print("Yes")
    5.     else:
    6.         print("No")
    7. else:
    8.     if t <= x < s:
    9.         print("No")
    10.     else:
    11.         print("Yes")
