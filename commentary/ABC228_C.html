「Final Day」(ABC228 C問題)

Difficulty : 214, solved : 4442/5045

問題文へのリンクはこちら

問題文

\(N\) 人の生徒が \(4\) 日間にわたる試験を受けています。

それぞれの日に行われる試験は \(300\) 点満点です。すなわち、 \(4\) 日間を通した試験の満点は \(1200\) 点です。

現在 \(3\) 日目までの試験が終わり、これから \(4\) 日目の試験が行われようとしています。 \(i(1 \le i \le N)\) 番目の生徒は \(j(1 \le j \le 3)\) 番目の試験で \(P_{i,j}\) 点獲得しました。

それぞれの生徒について、 \(4\) 日目の試験に上位 \(K\) 位に入っていることがあり得るかどうか判定して下さい。

ただし、 \(4\) 日目の試験後の生徒の順位は、その生徒よりも \(4\) 日間の合計点が高い生徒との人数に \(1\) を加えた値として定めます。

制約

  • \(1 \le K \le N \le 10^{5}\)
  • \(0 \le P_{i,j} \le 300 (1 \le i \le N, 1 \le j \le 3)\)
  • 入力は全て整数である。

解説

この問題は,各 \(i (1 \le i \le N)\) について, \(i\) 番目の生徒が \(4\) 日目の試験後に上位 \(K\) 位以内に入っているかを判定する問題です.

簡単のため, \(3\) 日目までに獲得した点数について, \(point[i](1 \le i \le N) = P_{i,1}+P_{i,2}+P_{i,3}\) と再び定義します.

それぞれの生徒が,一番順位を上げられるケースについて考えましょう.ズバリ,「各 \(i\) について,「生徒 \(i\) だけ \(300\) 点を獲得し, 残りの生徒が \(0\) 点を獲得した場合」がそのケースです。

このケースについて, 「生徒 \(i\) の点数に \(300\)を足したもの」が「 \(3\) 日目までに獲得した点数で上位 \(K\)番目の点数」以上であれば, \(4\) 日目に上位 \(K\) 位以内に入ることができます.

従って, \(point\) をあらかじめ降順にソートして,各 \(i (1 \le i \le N)\) について \(P[K] \le (P[i]+300) \) であるかを判定すればよいです.

計算量は,ソートの部分がボトルネックとなり, \(O(NlogN)\) となります.
